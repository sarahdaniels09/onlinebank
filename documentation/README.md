# Documentation — OnlineBank

This folder contains the documentation site for the OnlineBank project.

Preview
-------
Open `index.html` in your browser, or run a lightweight static server (recommended) to preview:

For Windows PowerShell (simple):

```powershell
# from repository root
Set-Location .\documentation
# If you have Python 3 installed
python -m http.server 8000
# then open http://localhost:8000
```

What's included
---------------
- `index.html` — single-page documentation
- `assets/css/style.css` — styling
- `assets/js/main.js` — small interactivity: smooth scroll, mobile menu, copy-to-clipboard
- `assets/img/` — placeholder screenshots (replace with production screenshots before CodeCanyon submission)
